steps for installation and setup
================================

1.install nodejs and nodejs path set in system environment variables.
2.download the attachment folder, go to folder path and select the folder right click select Extract ALL option.

============ Run the code in Visual Studio ==========================================

3.Open visual studio code select the project folder.
4.Inside visual studio open Integrated terminal give the following commands
==> npm install
==> npm run build  //this for production mode and minification files
==> npm start // this is for developmentMode.
